> common data structure implementation in javascript


## Install
[![NPM](https://nodei.co/npm/jal.png?mini=true)](https://nodei.co/npm/jal/)

```sh
$ npm install --save jal
```


## Usage

```js
//On NodeJS
var jal = require('jal');

var linkedList = new jal.SingleLinkedList();
linkedlist.add(1);

//On Browser
var linkedlist = new JAL.SingleLinkedList();
linkedlist.add(2);

```
